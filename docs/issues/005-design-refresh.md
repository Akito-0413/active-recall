# 005: デザイン改善

## 背景

MVPでは動作する学習フローを優先しました。今後は継続利用しやすいよう、見た目と操作感を整えたいです。

## やりたいこと

- ログ一覧・詳細・復習画面の視覚階層を見直す。
- モバイル表示を重点的に調整する。
- 空状態、成功状態、エラー状態の表現を整える。
- 学習に集中できるタイポグラフィと余白を再調整する。
- Custom GPTから保存されたログの表示をわかりやすくする。

## 受け入れ条件

- 主要画面がモバイルとデスクトップで破綻しない。
- 一覧から詳細、復習画面への導線がわかりやすい。
- 保存元、復習状態、作成日が自然に読み取れる。
- 学習内容が主役に見える。

## 注意点

- 学習アプリとしてミニマルで集中しやすいUIを維持する。
- マーケティングページ風の装飾や高密度ダッシュボード化は避ける。
