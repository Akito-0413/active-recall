# 004: Custom GPT作成支援

## 背景

Custom GPTは各ユーザーが作成・設定する必要があります。そこまでアプリ側でサポートできると、他ユーザーが使い始めやすくなります。

## やりたいこと

- Custom GPT作成手順をアプリ内またはドキュメントで案内する。
- ユーザーごとのOpenAPI schema差し替え・コピー手順を整備する。
- ユーザーごとのBearer APIキー設定手順を整備する。
- GPT Actionsのテスト手順を整備する。

## 受け入れ条件

- ユーザーが手順に従ってCustom GPTを作成できる。
- `servers.url` とAPIキーの設定箇所が明確に説明されている。
- 保存テストの成功確認手順がある。

## 注意点

- まずはドキュメントから始める。
- 必要になってからアプリ内UI化する。
- GPT Builder側のUI変更に依存しすぎない説明にする。
